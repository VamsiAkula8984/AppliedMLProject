# AppliedMLProject
Image Classification using CNN on Cifar10 Dataset
## About the Dataset
Name: CIFAR-10
Number of Images: 60000
Resolution: 32x32
Number of classes: 10 ["airplane","automobile","bird","cat","deer","dog","frog","horse","ship","truck"]
Train/Test Split: 50000/10000
